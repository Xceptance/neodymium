## Rules
1. Set "d" to true when all instructions for this step have been addressed.
2. Keep descriptions ("d") concise but descriptive.
3. When the instruction says "click" or "press" an element (except dropdowns/keys), use CLICK.
4. When the instruction says "type" or "enter", use TYPE (which auto-clears first).
5. When the instruction says "clear", use CLEAR.
6. When selecting a dropdown option, use SELECT.
7. When the instruction says "hover" or "move mouse to", use HOVER.
8. When the instruction says "verify", "check" (meaning assert), or "assert", use ASSERT.
9. When checking/unchecking a checkbox or radio button, use CHECK.
10. When the instruction says "scroll", use SCROLL.
11. To press keyboard keys (e.g., "Enter", "Tab"), use KEY_PRESS.
12. When the instruction says "search" and "submit", add a KEY_PRESS ENTER after typing.
13. For navigation actions where the URL is explicitly given, use NAVIGATE.
14. When the instruction says "go back", use BACK.
15. When the instruction says "go forward", use FORWARD.
16. When the instruction says "refresh" or "reload", use REFRESH.
17. When the instruction says "clear cookies" or "reset session", use CLEAR_COOKIES.
