Aura Defect Controls (Chaos Drawer)

Every sub-application contains a floating gear-wheel icon in the bottom-right corner of the page. Click it to open the Chaos drawer, where you can manually or programmatically inject visual and functional defects:

  • Inject Element Overlap: Forces buttons to slide absolute over primary header titles.
  • Inject Low Contrast: Overrides colors to near-invisible shades for testing contrast and a11y checks.
  • Inject Clipped Text: Caps description card heights to 60px with hidden overflow, hiding trailing text.
  • Inject Mobile Overlap: Squeezes multi-column grids horizontally without wrapping.
  • Inject Layout Shift: Displaces rows with off-center padding and margins to simulate misalignments.
  • Show Promo Banner: Mounts or unmounts a promotional banner on top of the layout to test optional steps.

Running Automated Tests

To compile and execute the pre-configured visual test suites locally, execute the following commands in your workspace root:

mvn test -Pjunit-5 -Dtest=com.xceptance.neodymium.ai.Aura*Test

The test suites automatically spin up the embedded server, load the test applications, and perform various visual AI assertions, visual dHash baseline replays, accessibility audits, and AJAX/HTMX checks.
